# TA-432 releases

Installer downloads for TA-432, Tesserae Audio's 440 -> 432 Hz (or custom ratio) pitch converter.

See [tesseraeaudio.github.io/website](https://tesseraeaudio.github.io/website/) for details, or check the [Releases](../../releases) page to download.
